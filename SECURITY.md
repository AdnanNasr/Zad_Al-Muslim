## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not open a public issue. Instead, report it responsibly by following these steps:

1. Send an email with the details of the vulnerability to `zedxxxxx317@gmail.com`.
2. Please include a clear description of the issue and steps or a proof-of-concept (PoC) to reproduce it.
3. We will acknowledge receipt of your report within `48 hours`.
4. We will keep you updated on our progress as we work to patch the vulnerability and release a secure update.
